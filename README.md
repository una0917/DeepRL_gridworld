# Grid World Environment - 深度強化學習作業

## 專案概述

這是一個基於 **Flask** 和 **Web UI** 的強化學習環境實現，用於解決經典的網格世界（Grid World）問題。本專案通過值迭代算法（Value Iteration）和策略評估（Policy Evaluation）來找出最優路徑。

## 功能說明

### 核心算法
- **值迭代（Value Iteration）**：計算每個狀態的最優價值函數
- **最優策略推導**：基於值函數找出每個狀態的最優行動
- **路徑規劃**：從起點利用最優策略導航至目標點

### 主要功能
1. **初始化網格** - 設定網格大小（5×5 至 9×9）
2. **設置環境**
   - 點擊選擇**起點**（綠色）
   - 點擊選擇**目標點**（粉紅色）
   - 點擊選擇**障礙物**（黑色）
3. **執行策略評估** - 計算最優價值和策略矩陣（不包含路徑）
4. **執行值迭代** - 計算最優價值、策略矩陣並顯示最優路徑

## 使用方法

### 環境要求
- Python 3.7+
- Flask

### 安裝與運行

```bash
# 安裝依賴
pip install flask

# 運行應用
python app.py

# 訪問應用
# 在瀏覽器中打開 http://localhost:5000
```

### 操作步驟

1. **初始化**
   - 輸入網格大小（例如 5）
   - 點擊「Initialize Grid」按鈕

2. **設置環境**
   - 點擊一個格子設置**起點**（顯示綠色高亮）
   - 再點擊一個格子設置**目標點**（顯示粉紅色高亮）
   - 點擊其他格子設置**障礙物**（顯示黑色）
   - 可重複點擊切換設置或清除

3. **運行算法**
   - 點擊「2. Run Policy Eval」- 查看策略和價值矩陣
   - 點擊「3. Run Value Iter」- 查看路徑規劃結果

4. **結果展示**
   - **策略矩陣**: 顯示每個格子的最優行動方向（↑↓←→）
   - **價值矩陣**: 顯示每個格子到目標的預期價值
   - **路徑高亮**: 黃金色突出最優路徑

## 算法詳細說明

### 值迭代算法流程

```
初始化所有狀態價值 V(s) = 0

重複直到收斂：
  對每個狀態 s：
    max_value = -∞
    最優行動列表 = []
    
    對每個可能行動 a：
      計算下一狀態 s' 的坐標
      如果 s' 有效（在網格內且不是障礙物）：
        r = 獎勵 + γ × V(s')
        如果 r > max_value：
          max_value = r
          最優行動列表 = [a]
        否則如果 r ≈ max_value：
          最優行動列表.append(a)
    
    V(s) = max_value
    policy(s) = 最優行動列表
  
  檢查收斂條件（delta < THETA）
```

### 狀態轉移
- **有效移動**: 在網格範圍內且非障礙物
- **無效移動**: 撞牆或碰到障礙物，保持原位
- **獎勵結構**
  - 每步移動: -1
  - 到達目標: +10
